Detailed Description

A general joystick controller.

The joystick controller expects a quantizationlevel struct as input. This is to indicate the max and min y and x direction of the joystick. Example: If x goes from 0 to 255, it expects input struct with x_max = 255, x_min = 0. If y goes from 0 to 255, it expects the same for y. It also expect a uint8_t threshold, given in decimal. This threshold is used for detecting direction This must be initialized through the Initialize function, due to singleton
