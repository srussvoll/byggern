\subsection{Detailed Description}
Keeps track of the highscore and stores it to the E\-E\-P\-R\-O\-M. 

The \hyperlink{class_highscore_1_1_highscore}{Highscore} class keeps track of the highscores, and provides a method for saving them to the E\-E\-P\-R\-O\-M for persistent storage.
